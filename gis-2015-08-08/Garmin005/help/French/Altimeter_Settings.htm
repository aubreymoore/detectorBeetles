Paramètres de l'altimètre

Etalonnage automatique : permet d'étalonner l'altimètre automatiquement chaque fois que vous allumez l'appareil. L'appareil continue à étalonner automatiquement l'altimètre tant que des signaux GPS sont acquis.

Mode Baromètre : l'option Altitude variable permet au baromètre de mesurer les changements d'altitude tandis que vous vous déplacez.L'option Altitude fixe suppose que l'appareil est stationnaire à une altitude fixe, la pression barométrique n'étant donc censée changer qu'en fonction de la météo.

Enregistrement tendances pression : permet de définir la façon dont l'appareil enregistre les données de pression.L'option Enregistrer toujours peut être utile lorsque vous recherchez des fronts de pression.

Type de tracé : permet d'enregistrer les variations d'altitude sur une période ou une distance donnée, la pression barométrique ou les changements de pression atmosphérique sur une période donnée.

Etalonner l'altimètre : permet d'étalonner manuellement l'altimètre barométrique si vous connaissez l'altitude ou la pression barométrique correcte.
