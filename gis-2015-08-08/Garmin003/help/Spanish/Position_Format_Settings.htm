Configuración del formato de posición

NOTA: no cambies el formato de posición ni el sistema de coordenadas del datum del mapa a menos que estés utilizando un mapa o carta que especifique un formato de posición diferente.

Formato de posición: establece el formato de posición en el que se muestra una lectura de ubicación.

Datum del mapa: establece el sistema de coordenadas con el que está estructurado el mapa.

Esferoide del mapa: muestra el sistema de coordenadas que está utilizando el dispositivo. El sistema de coordenadas predeterminado es WGS 84.
