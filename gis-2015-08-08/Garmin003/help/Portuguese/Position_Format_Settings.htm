Definições do formato da posição

NOTA: Não deverá alterar o formato da posição ou o sistema de coordenadas dos dados de referência do mapa, exceto se estiver a utilizar um mapa ou uma carta que especifique um formato da posição diferente.

Formato da posição: Define o formato da posição no qual a leitura de local aparece.

Dados de referência do mapa: Define o sistema de coordenadas no qual o mapa é estruturado.

Esferóide do mapa: Apresenta o sistema de coordenadas que o dispositivo está a utilizar. O sistema de coordenadas predefinido é WGS 84.
